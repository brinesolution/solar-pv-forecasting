## Structure

- `sections/`: manuscript body sections.
- `figures/diagrams/`: regenerated clean architecture, workflow, split, and
  inference diagrams.
- `figures/results/`: selected result figures copied from the active project.
- `figures/diagnostics/`: selected diagnostic figures copied from the active
  project.
- `tables/`: LaTeX tables, `references_audit.csv`, and metric audit outputs.
- `macros/`: reusable manuscript constants and notation.
- `references.bib`: local ordered reference entries are retained for internal
  continuity but are not cited/nocited in the compiled bibliography until
  verified. The compiled bibliography currently uses the PV forecasting,
  XGBoost, clear-sky, Open-Meteo, and NREL entries.
- Upload ZIP prepared one level above this folder:
  `ieee_access_latexdraft_overleaf_upload.zip`. The ZIP excludes `build/`.

## Important Pre-Submission Tasks

1. Replace the four dummy authors, affiliations, emails, biographies, DOI, and
   funding footnote.
2. Verify every local ordered PDF reference in `tables/references_audit.csv`.
3. Rerun `assets/recompute_manuscript_metrics.py` if the active dataset,
   metadata, or saved models change, then refresh metric tables and result
   figures.
4. If the official IEEE Access `ieeeaccess.cls` is supplied later, compare it
   against the local compatibility wrapper before replacing it.
5. Compile once in Overleaf and check for unresolved references, missing
   citations, figure scaling, and overfull boxes.
